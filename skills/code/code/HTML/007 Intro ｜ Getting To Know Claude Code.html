    Intro

    In this course, I'm going to be primarily using Claude Code as my agent for driving the lessons, demonstrating things, and showing you how agents in general work.

    I'm doing this because it's a very popular harness that you may have used yourself, and it's the one that I personally use the most right now. However, this course is harness agnostic - you can choose whatever harness you like, as long as it runs in a CLI.

    For folks who've never used a coding harness before, or who just want a bit more out of Claude Code, I've included a primer section so that you can get the basics before we get started.

    If you haven't installed Claude Code yet, you can find the setup instructions in the official docs.

    It will give you a nice grounding of all the things you'll need to know about Claude Code before getting started with the course.

    And if you're experienced with Claude Code, you'll probably still pick up a few tips - I'm not just covering the basics. There are a couple of advanced things in there too.

    So nice work, and I will see you in the next one!
