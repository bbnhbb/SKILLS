    Sandcastle

    When you first start working with AFK agents, there's a learning curve. You need time to interact with the agent, check its progress, and refine your setup.

    This means we need two different ways to run agents:

    1. HITL (Human-In-The-Loop) - for learning and testing
    2. AFK (Away-From-Keyboard) - for production use

    Let's start with the HITL approach so you can see what's happening.

    Understanding Sandcastle

    Sandcastle is a tool that lets you run agents inside different sandbox environments. Instead of being locked into one provider, you can:

    • Use Docker for local testing
    • Use Podman as an alternative
    • Use Vercel for remote isolation
    • Or configure your own setup

    The key idea is sandbox agnostic - swap providers without changing your agent code.

    Setting Up Your Interactive Agent

    For this exercise, we're going to start without a sandbox at all. This keeps things simple while you learn how the agent works.

    Here's what we're using:

    • interactive function from Sandcastle
    • claudeCode agent (Claude Opus 4.6)
    • noSandbox - no sandbox environment
    • prompt.md file containing your agent instructions
    import { claudeCode, interactive } from "@ai-hero/sandcastle";

    import { noSandbox } from "@ai-hero/sandcastle/sandboxes/no-sandbox";

    

    await interactive({

      agent: claudeCode("claude-opus-4-6"),

      sandbox: noSandbox(),

      promptFile: "./.sandcastle/prompt.md",

    });

    Why No Sandbox Yet?

    Setting up a sandbox environment has some overhead. By running without one first, you can:

    • Focus on crafting the right prompt
    • Understand how the agent behaves
    • Watch the agent carefully and see what works
    • Optimize your setup before adding complexity

    Once you understand the agent's behavior, you can add Docker or another sandbox provider.

    Steps To Complete

    • Check out the Sandcastle repo

    Take a look at the documentation and examples. This will help you understand how Sandcastle works and what it can do.

    • Read the file at .sandcastle/interactive.ts

    This file shows how to set up an interactive HITL agent using Sandcastle. You'll use this approach in the next exercises to test and refine your AFK agent setup.
