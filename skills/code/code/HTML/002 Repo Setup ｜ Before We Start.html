    Repo Setup

    Before you start building, you need to get the course project running locally. This involves cloning the repository, installing Node.js and pnpm, seeding the database, and setting up an agent.

    The good news is that the setup is straightforward - just follow the steps below in order.

    Steps To Complete

    Clone the Repository

    • Clone the AI Hero Dev Cohort 004 project to your local machine

    Go to the repository, press the code button, select HTTPS, and copy the URL.

    git clone https://github.com/ai-hero-dev/cohort-004-project.git

    Then navigate into the cloned directory.

    Install Node.js

    • Download and install Node.js from https://nodejs.org/en/download

    Choose either version 22 or version 24 - both will work fine.

    Select the installer for your operating system and follow the installation instructions.

    • Verify Node.js is installed by running the following in your terminal
    node -v

    You should see a version number printed back.

    Set Up the Package Manager

    • Enable pnpm by running this command in your terminal
    corepack enable

    This installs pnpm, the package manager used by this project. pnpm is faster than npm and saves disk space.

    • Verify pnpm is installed by running
    pnpm -v

    You should see a version number printed back.

    Install Dependencies

    • Install the project dependencies
    pnpm install

    This downloads all the required packages from the NPM registry into your node_modules folder.

    Seed the Database

    • Seed the database with dummy data
    pnpm db:seed

    This creates the database tables and populates them with sample data. The project is a course platform, so the database includes users, courses, categories, quizzes, enrollments, lesson progress records, quiz attempts, video watch events, and more.

    • Start the development server
    pnpm dev

    The development server should start successfully and run on localhost:5173 or similar.

    • Visit the URL in your browser

    You should see a Cadence video platform. Click on Courses to see the available courses.

    There's also a dev UI at the bottom of the page where you can log in as different users.

    • Stop the development server by pressing CTRL-C when you're ready to move on

    Install Your Agent

    • Install an agent if you don't already have one

    The work you'll do in this course is designed to work with any CLI-based agent. Visit the setup documentation for your agent of choice and follow the installation instructions.

    For Claude Code, visit the Claude Code setup docs.

    • Run your agent in your terminal
    claude

    This should open your agent. You may need to log in with your API key or subscription.

    • Test that your agent is working

    Type hello and it should reply with a message.

    • You're ready to start the course!

    If you hit any issues during setup, head to the AI Hero Discord and ask for help.
